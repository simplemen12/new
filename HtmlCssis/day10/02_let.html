자바스크립트 변수 사용

  • 값을 담고 있는 메모리 공간의 이름
  • java에서는 자료형 변수명;
  • 자바스크립트는 변수는 선언하는 역할만 함. 자료형이 없다
  • var, let(ES6 var는 권장하지 않음), const,
  • let자바에서 사용되는 지역변수의 개념
  • let 중복선언 안됨 재할당 가능 let name = 'hong'; let name; 'x' name = 'kim';
  • const 중복선언 안됨 재할당 안됨 const name = 'hong'; name = 'kim'; 불가능
  • 스코프(scope 사용범위) { } 안에서만 사용가능 스코프를 벗어나면 재선언 가능
  • const 고정된 값을 사용하는 규칙 참조 객체 엘리먼트 정의시 사용
